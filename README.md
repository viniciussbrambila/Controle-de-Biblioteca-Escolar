## Regras de Negócio
1. Não é permitido cadastrar livros com o título vazio.
2. Não é permitido cadastrar livros com a quantidade negativa.
3. Não é possível realizar o empréstimo de um livro se a quantidade disponível for zero.
4. O sistema deve registrar tanto a saída (empréstimo) quanto a entrada (devolução) do livro.
5. O sistema deve permitir consultar quais livros estão emprestados e quais alunos estão com empréstimos em aberto.
